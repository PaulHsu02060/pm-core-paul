# PM-Core Roadmap

## 北極星
全 HTML 單一入口的 PM 系統，讓 PM 少漏一件事（正確數據、減少疏忽）。資料主權在 PM-Core（localStorage→雲端），不依賴外部 Sheet。目標：月中，雛型完整且能正常使用。不趕半成品（前兩套系統的教訓）。

## 系統三層結構（既有架構，已成形）
1. 主頁 Dashboard（跨專案總覽）
2. 專案層（每專案：Dashboard 分頁 + Gantt 分頁）
3. 跨專案分析頁

## 六大塊現況（2026-06-05 盤點）
- 完整：核心計算引擎、Gantt、月曆、週報、PDCA、DB/雲端同步、資料模型
- 半成品：主頁 Dashboard（缺跨專案聚合）、專案頁（缺資訊條/分頁）、Auth（純前端裝飾，無真安全層）
- 不存在：跨專案分析頁
- bug：DELAYED 漏報（已修 cc0efb7，待回家 node 驗證）
